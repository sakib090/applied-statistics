Statistical analysis coursework - distributions, MLE, hypothesis testing and linear regression using Python
# Applied Statistics — ECS764P (MSc Data Science, QMUL)

Statistical analysis coursework completed as part of the MSc Data Science 
programme at Queen Mary University of London.

## Coursework 1a
Probability distributions and convolutions. Computing and visualising the 
sum of chi-squared distributions, sample mean distributions using Gamma 
distributions, and Maximum Likelihood Estimation on Dow Jones Industrial 
Average daily returns.

## Coursework 1b
Theoretical exercises covering normal distributions, sample means, Bernoulli 
sums, and probability measure scaling.

## Coursework 2
Open-ended statistical investigation into dependency between two real-world 
datasets. Includes joint distribution analysis, hypothesis testing for linear 
dependence, linear regression, residual analysis, and KS-test for normality.

## Tech Stack
- Python
- NumPy / SciPy
- pandas
- Matplotlib
- Jupyter Notebook
